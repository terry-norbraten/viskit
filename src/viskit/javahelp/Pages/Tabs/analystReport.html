Analyst Report

The Analyst Report tab allows analysts to iteratively add description statements, questions and conclusions regarding their simulation.


Screen Capture of Analyst Report Tab.

There are eight sub-tabs in this main tab. A short description of each follows.

  1. Heading — Specifies the report heading.
  2. Executive Summary — Contains free-form HTML that is the executive summary of the document.
  3. Simulation Location — Contains free-form HTML that relates to the location of the simulation run.
  4. Simulation Configuration — Contains free-form HTML and optional entity table relating to the configuration of the simulation.
  5. Entity Parameters — Contains free-form HTML and optional entity table relating to the entity parameters.
  6. Behavior Definitions — Contains free-form HTML and optional computer-generated behavior descriptions and images.
  7. Statistical Results — Contains free-form HTML and optional statistical tables relating to the statistical results.
  8. Conclusions and Recommendations — Contains free-form HTML providing the conclusions and recommendations for future work.
